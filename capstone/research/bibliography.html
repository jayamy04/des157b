Capstone Project Research - Annotated Bibliography

Locate at least three sources. Sources may include books, articles, websites and videos. Write the sources in a list using consistent citation formatting (preferably MLALinks to an external site.). Note: The best option is to design and code this assignment using HTML and CSS (saved as bibliography.html in your capstone/research folder, but we will accept a well-designed and web-friendly PDF, or a Google Doc with attention to design. Add a paragraph summary for each source (approximately 100 words), including the main focus or purpose of the work, an evaluation of its usefulness to your project, a summary of the author's conclusions and your own conclusions. Note: you may find it helpful to add to this document as your project develops and you find more resources. This is a great artifact for a portfolio piece about this project.
